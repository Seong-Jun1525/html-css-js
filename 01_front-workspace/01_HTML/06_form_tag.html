숫자와 관련된 input 태그의 타입

type="number"

  • 숫자에 대한 값만 작성 가능한 텍스트 상자
  • 스핀 박스가 표시됨(* 스핀박스 : 위/아래 화살표 버튼)

type="range"

  • 슬라이드 바를 통해 숫자 지정 가능

날짜 및 시간 관련된 input 태그의 타입

  • 날짜 : date
  • 월 : month
  • 주 : week
  • 시간 : time
  • 날짜 + 시간 : datetime-local

라디오 버튼과 체크박스 input 태그의 타입

라디오 버튼

  • type="radio"
  • 여러 값들 중 하나의 값만 선택하고자 할 때
  • name을 동일하게 줘야지 하나의 값만 선택할 수 있다
  • 전달하고자 하는 값은 value로 설정해줘야함

체크박스

  • type="checkbox"
  • 여러 값들 중 여러 값들을 선택할 수 있도록 하고자 할 때
  • 선택된 체크박스의 value 속성 값이 전달됨 반드시 value 속성 값을 지정해줘야함

그 외 input 태그의 타입

type="color"

  • 색상을 선택할 수 있는 타입

type="file"

  • 파일을 첨부

type="hidden"

  • 사용자에게 입력받지 않고 화면에 출력하지도 않으면서 서버에 넘길 데이터 값이 있는 경우 사용

버튼 관련 타입

  • submit : 제출
  • reset : 사용자 입력 데이터 초기화
  • button : 일반적인 버튼

select 와 option 태그 사용

  • 사용자가 직접 입력하는 것이 아니라 제시한 여러 개의 값들 중 선택하도록 하고자 할 경우
  • 제출 시 선택된 option 태그의 값이 전달됨
    • 각 option 태그의 value 속성이 지정되지 않은 경우 내용부분의 값이 전달됨
    • value 속성이 지정되어 있는 경우 해당 value 속성 값이 전달됨


datalist 태그

  • 사용자에게 직접 값을 입력받거나 select 태그처럼 목록을 통해 선택받을 수 있는 태그
