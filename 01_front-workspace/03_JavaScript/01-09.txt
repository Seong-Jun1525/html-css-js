- 자바스크립트는 [ECMA] 표준을 따르는 웹 기술이다.
- 개발자 도구를 보기 위해서는 [F12] 또는 [검사] 를(을) 통해 볼 수 있다.
- 자바스크립트 코드를 HTML 문서 내에서 사용하기 위해서는 [script] 태그가 필요하다.

- 브라우저 화면에 데이터를 출력하고 싶을 때 : document.write()
- 특정 요소에 데이터를 출력하고 싶을 때 : innerHTML innerTEXT
- 메시지창으로 데이터를 출력하고 싶을 때 : alert 
- 개발자 도구의 콘솔창에 데이터를 출력하고 싶을 때 : console.log()
