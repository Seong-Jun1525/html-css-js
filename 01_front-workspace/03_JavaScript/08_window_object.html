Window 객체

window 객체는 js에서 최상위 객체
크게 BOM과 DOM으로 나누어 짐

  • BOM (Browser Object Model)
    • location(주소창 관련)
    • screen(창 관련)
    • navigator(브라우저 관련)
    • history(페이지 관련 기록)
  • DOM (Document Object Model)
    • document

[window.]open("url 주소" "창 이름", "창 특징")

[window.]setTimeout(함수, 시간)

설정한 시간 후에 함수의 내용을 실행(한번만)

[window.]setInterval(함수, 시간)

설정한 시간마다 함수를 실행

[window.]onload

화면에 윈도우 객체의 로드가 완료되면 설정되어 있는 함수를 실행
윈도우 객체의 로드 완료 → 모든 태그가 화면에 표시된 상태
