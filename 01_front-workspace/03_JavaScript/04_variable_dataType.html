변수와 자료형

변수 선언 시 var, let, const라는 키워드를 사용해서 선언 가능

* 변수 선언 방법

  • var : 중복 선언 가능, 재할당 가능
  • let : 중복 선언 불가능, 재할당 가능
  • const : 중복 선언 불가능, 재할당 불가능, 선언 시 초기화 해줘야 함

자바스크립트의 자료형

  • 숫자 : Number
  • 문자열 : String
  • 논리값 : Boolean
  • 초기화되지 않은 것 : undefined
  • 값이 없는 것 : null (Object 타입)
  • 객체/배열/함수 : Object

변수 명명 규칙

  1. 일반적으로 영어(대/소문자)를 사용하고, 문자와 숫자를 모두 사용 가능
  2. 숫자로 시작할 수 없음
  3. 예약어(키워드)를 변수명으로 사용할 수 없다
  4. 특수문자 _ 또는 $만 사용 가능

표기법 (Naming Type)

  1. camelCase(카멜표기법)
    소문자시작 다른 단어가 시작될 때 마다 대문자로 구분
    • userId
    • userEmail
  2. snake_case
    단어를 _로 구분
    • user_id
    • user_email
  3. PascalCase : C# 또는 MS 계열의 언어에서 사용되는 표기법
    모든 단어의 시작을 대문자로 구분
    • UserId
    • UserEmail
