  3. document.write("화면에 출력");
  4. 선택된요소.innerHTML = "요소에 출력 내용"
    선택된요소.innerTEXT = "요소에 출력 내용"

데이터를 입력 받는 구문(변수에 저장)

  1. 변수 = [window.]confirm("질문 내용");
  2. 변수 = [window.]prompt("질문 내용");
  3. 변수 = 선택된 요소.속성; (id, className, innerText, ...)
  4. 변수 = 선택한 input요소.value

[window.]confirm("질문 내용");

confirm 호출 시 "질문 내용", 확인/취소 버튼이 존재하는 알림창
확인 버튼 클릭 시 true, 취소 버튼 클릭 시 false를 반환


[window.]prompt("질문 내용");

prompt 호출 시 "질문내용", 입력할 수 있는 "텍스트 상자", 확인/취소 버튼이 보여지는 알림창
확인 버튼 클릭 시 텍스트 상자에 입력된 값을 반환
취소 버튼을 클릭 시 null 반환
