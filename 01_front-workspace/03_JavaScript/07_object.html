객체

객체 선언 및 사용

객체는 {}를 사용해서 생성,
{} 안에 필요한 속성들을 정의한다.

형태 → 속성명(key): 속성값(value)

객체를 파일로 저장할 때 확장자를 .json으로 설정
JSON형식이라고 표현하기도 한다. 

        변수 = {
            속성명: 속성값,
            속성명: 속성값,
            속성명: 속성값,
            ...
        }

객체에서의 반복문

객체의 함수 속성 : 메서드

객체의 속성 중 함수 형태의 속성을 메서드라고 부름

in, with 키워드

in : 객체 내의 해당 속성이 있는지 확인해주는 키워드
true : 속성이 있다 false: 속성이 없다

with: 객체의 속성 접근 시 객체 별로 속성을 제시하는 것을
생략할 수 있게 해주는 키워드(코드를 간결하게 해줌)
