배열

자바스크립트에서는 변수 선언 시 별도의 자료형 지정 없이
어떤 자료형이든 배열에 담을 수 있다.
따라서 크기의 제약이 없다. (자바의 컬렉션과 유사)

배열의 선언

[] 사용하거나 new 연산자의 형태로 선언할 수 있음

배열 관련 메서드

1) 배열명.indexOf(찾고자하는 값);

배열에서 해당 값이 위치한 인덱스 반환
해당 값이 있으면 인덱스 반환(0 ~)
해당 값이 없으면 -1 반환

2) 배열명.concat(배열, 배열, ...);

배열에 여러 개의 배열을 결합하고자 할 때 사용
합쳐진 배열을 반환
기존 배열에 영향을 주지 않는다

3) 배열명.reverse();

배열에 담긴 값을 역순으로 바꿔주는 기능
기존 배열에 영향을 준다
