배열

자바스크립트에서는 변수 선언 시 별도의 자료형 지정 없이
어떤 자료형이든 배열에 담을 수 있다.
따라서 크기의 제약이 없다. (자바의 컬렉션과 유사)

배열의 선언

[] 사용하거나 new 연산자의 형태로 선언할 수 있음

배열 관련 메서드

1) 배열명.indexOf(찾고자하는 값);

배열에서 해당 값이 위치한 인덱스 반환
해당 값이 있으면 인덱스 반환(0 ~)
해당 값이 없으면 -1 반환

2) 배열명.concat(배열, 배열, ...);

배열에 여러 개의 배열을 결합하고자 할 때 사용
합쳐진 배열을 반환
기존 배열에 영향을 주지 않는다

3) 배열명.reverse();

배열에 담긴 값을 역순으로 바꿔주는 기능
기존 배열에 영향을 준다

4) 배열명.sort()

배열 내의 값을 오름차순으로 정렬시키는 함수
기존 배열에 영향을 준다

5) 배열에 값을 추가/제거 - 1

추가 : 배열명.push(추가할 값);

배열의 맨 뒤에 값을 추가하고, 배열의 크기를 반환
기존 배열에 영향을 준다

제거 : 배열명.pop()

배열의 맨 뒤에 값을 제거하고, 그 값을 반환
기존 배열에 영향을 준다

6) 배열의 값을 추가 제거하는 - 2

추가 : 배열명.unshift(값);

배열의 맨 앞에 값을 추가하고, 배열의 크기를 반환
기존 배열에 영향을 준다

제거 : 배열명.shift();

배열의 맨 앞에 값을 제거하고, 그 값을 반환
기존 배열에 영향을 준다

7) 배열에서 추출하기

배열명.slice(시작인덱스, 끝인덱스);

배열에서 시작인덱스 위치부터 끝인덱스 전까지 배열의 값들을 추출한다
기존 배열에 영향을 주지 않는다

배열명.splice(시작인덱스, 제거할 개수[, 추가할 값])

배열에서 시작인덱스 위치부터 개수만큼 추출하여 제거
(+ 데이터를 추가할 수 있도록 해주는 함수)
기존 배열에 영향을 준다

8) 배열을 문자열로 변환하기

배열명.toString();

배열명.join([구분자]);

배열의 각 인덱스에 담긴 값을 하나의 문자열로 반환해주는 함수
