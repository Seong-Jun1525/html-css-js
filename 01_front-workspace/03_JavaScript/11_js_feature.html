* 호이스팅(hoisting)

모든 변수 선언문이 코드의 최상단으로 이동되는 것처럼 느껴지는 현상
var 타입에서 확인되고, let이나 const 타입에서는 호이스팅 일어나지 않는 것처럼 동작되게 해줌

* 스코프(변수의 범위)

변수의 사용 가능 범위
- var : functional scope. 변수가 선언된 함수 영역 내에서 사용 가능
- let, const : block scope. 변수가 선언된 블록 내에서만 사용 가능



* 전역 변수, 지역 변수

- 전역 변수 : 특정 함수 영역 밖에 선언된 함수(어디서든 사용 가능)
- 지역 변수 : 특정 범위 내에 선언된 변수

* 배열 관련 메서드

배열.map(함수)

배열 내의 모든 요소 각각을 함수의 결과 값으로 새로운 배열을 만들어서 반환



배열.filter(함수)

배열 내의 요소들을 주어진 함수에 의해 조건을 만족하는 항목만 모아 새로운 배열 반환



배열.reduce(함수)

배열 내의 요소들을 주어진 함수를 실행하고 하나의 결과 값을 반환
