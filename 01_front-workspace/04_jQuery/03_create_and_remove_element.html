요소 생성 및 제거

* 삽입 관련 메서드

선택된 요소를 기준으로 새로운 요소를 추가하는 메서드 

        $(A).append(B) : A요소 내에 뒷 부분에 추가 (자손/하위)
            - 이미 요소가 있을 수 있음 그 뒤에 즉 마지막에 B요소를 추가
        $(A).prepend(B) : A요소 내에 앞 부분에 B요소를 추가 (자손/하위)
            - 이미 요소가 있을 수 있음 그 앞에 즉 맨 처음에 B요소를 추가
        $(A).after(B) : A요소 다음에 B요소를 추가 (동위)
        $(A).before(B) : A요소 전에 B요소를 추가 (동위)

* 제거 관련 메서드

  • $("선택자").empty() : 선택된 요소의 하위 요소를 제거해주는 메서드
  • $("선택자").remove() | detach() : 선택된 요소를 제거한 후 해당 요소를 반환해주는 메서드
    • .remove() : 제거된 요소의 이벤트를 가져오지 않음
    • .detach() : 제거된 요소의 이벤트까지 가져옴
